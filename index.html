The following table defines the implicit native semantics and corresponding default implicit ARIA semantics that apply to SVG 1.1 elements. Each language feature (element) in a cell in the first column implies the ARIA semantics (role, states, and/or properties) given in the cell in the second column of the same row. The third column defines restrictions as to what WAI-ARIA semantic (role, state, or property) may or may not apply.

Documents must not use any role values with elements in the following table other than those listed for that element in the second column unless allowed by the restrictions defined in the third column.
